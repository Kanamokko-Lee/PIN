⚙️ Gist 接続設定

GitHub GistのRAW URLとPersonal Access Tokenを入力してください。
Tokenは gist スコープのみで十分です。

📖 設定手順：
1. gist.github.com で新規Gistを作成
   ファイル名: airports.json、内容: ダウンロードしたJSONを貼り付け
2. Gistを Secret または Public で保存
3. 「Raw」ボタンのURLをコピーして上に貼り付け
4. github.com/settings/tokens でトークン発行
   スコープ: gist にチェックするだけでOK
5. GistのURLから長い英数字部分をGist IDとして入力
